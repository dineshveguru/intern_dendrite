# Dendrite Intern Task

## Problem Statement

Develop a simple web app to search music and create playlist, similar to Spotify

## Technologies Used

- **React Js with Typescript**
- **Bootstrap 5**
- **KeyCloak** for Authentication
- **Redux** for state Management
